2 The xtpipes Script

The translation of a file is driven by a script that determines the transformations to be applied to the diffrent fragments of the input data. The name of the script may be found in different locations.

  • A xtpipes processing instruction within the input
  • Command line option ‘-s’
  • A .map file that checks the data characteristics to determine the script. Currently available only to input files, not to input strings.
