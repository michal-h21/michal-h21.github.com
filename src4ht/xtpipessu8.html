1.8 Output File

The output file name mentioned in the input command line ‘-o file.out’ is employed in the following cases.

  • Within sax commands without ‘name’ argument
  • Within the return instruction, if none of the sax commands wrote into the file. The returnToFile field is used for book keeping of whether a sax command used the file earlier for an output target.
