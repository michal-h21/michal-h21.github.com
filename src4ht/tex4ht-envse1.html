1 Outline of Scripts

Note: The ‘-c...’ calls are executed in the order of the prsence of the scripts within tex4ht.env. Hence, place the xtpipe translations before validations.

<..tex4ht.env-unix..>
 <.unix copyright.>%
 <.unix opening comment.>%
 <.unix note fonts.>%
 <.unix texmf dir.>%
 <.unix tfm fonts.>%
 <.unix htf fonts.>%
 <.unix G script.>%
 <.unix tex4ht.fls.>%
 <.unix t4ht commands.>%
 <.xtpipes: unix java help.>
 <.xtpipes: unix emacspeak.>
 <.xtpipes: unix jsml.>
 <.xtpipes: unix docbk.>
 <.xtpipes: unix xhtml.>
 <.xtpipes: unix xhtmml.> before validate
 <.xtpipes: unix oo.>%
 <.unix validate.> %
 <.unix openoffice.>%
 % end of file
